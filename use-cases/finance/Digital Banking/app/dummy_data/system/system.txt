    -Action Tracking: Records a variety of events from simple LOGIN_SUCCESS to administrative changes like LIMIT_INCREASE_REQUESTED.
    -Severity Levels:
       -INFO: Standard user activity (Logins, profile views).
       -HIGH: Sensitive changes that impact financial security (Password changes, credit limit requests, new beneficiaries).
    -Traceability: Every log includes a timestamp and ip_address, allowing for the reconstruction of a user's session in case of a security incident.
    -Outcome Mapping: Uses the status field to distinguish between completed actions and thwarted attempts (e.g., LOGIN_FAILED).
